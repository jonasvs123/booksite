# Booksite (Flask minimal)


## Quick start


1. **Create a virtual environment**
- macOS/Linux:
```bash
python -m venv .venv
source .venv/bin/activate
```
- Windows (PowerShell):
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```


2. **Install dependencies**
```bash
pip install -r requirements.txt
